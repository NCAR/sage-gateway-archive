DataTables README

DataTables is a plug-in for the jQuery Javascript library that provides pagination, instant search and multi-column ordering.
https://www.datatables.net/

The library was created using the DataTables download builder at https://www.datatables.net/download/.

The following options were selected:

  • jQuery - No jQuery
  • Styling - Bootstrap
  • Styling - No styling library
  • DataTables - DataTables
  • Extensions - Buttons
  • Extensions - HTML5 export
  • Extensions - Print view
  • Packaging options - Minify
  • Packaging options - Single file
  • Packaging options - Local files

Include the following in your HTML to load the required Javascript and CSS for the options selected above.

<link rel="stylesheet" type="text/css" href="/js/datatables/datatables.min.css"/> <script type="text/javascript" src="/js/datatables/datatables.min.js"></script>
